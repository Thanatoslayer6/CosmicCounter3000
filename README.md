# Cosmic Counter

Discord bot that solves problems

# Instructions

1.) Install node dependencies run

```
yarn install
```

1.) Create `.env` file from the project directory

2.) Add token and client id e.g.

```toml
TOKEN=abcdefgh
CLIENT_ID=abcdefgh
GUILD_ID=1abcddefgh
BING_COOKIE=`your bing authentication cookie the '_U' field`
```

3.) Run `index.js`
